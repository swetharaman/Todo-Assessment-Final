# Full Stack TODO Application

This is a full-stack TODO application built with the latest technologies: **Angular 21** for the frontend and **.NET 10** for the backend API. 

## Prerequisites
To run this project locally, ensure you have the following installed:
* **Node.js** (v24+) & **npm**
* **.NET 10 SDK**
* **Angular CLI** (v21+)

---

## How to Run the Backend
1. Open a terminal and navigate to the `TodoListApplication` folder.
2. Run `dotnet restore`
3. Run `dotnet run` (The API will run in-memory on https://localhost:7173)

## How to Run the Frontend
1. Open a terminal and navigate to the `TodoWeb` folder.
2. Run `npm install`
3. Run `npm start` (or `ng serve --open`)
4. The application will open at http://localhost:4200
